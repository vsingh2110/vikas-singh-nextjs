### Step 5: Add Recipients Using BCC

âš ï¸ **VERY IMPORTANT: Always use BCC!**

**Why BCC?**
- Protects client privacy - recipients cannot see each other's email addresses
- Looks professional - each recipient thinks they received a personal email
- Required for GDPR/privacy compliance
- Prevents "Reply All" disasters

**How to add BCC recipients:**

1. Click on **BCC** field (next to "To" field)
2. Add recipient email addresses separated by commas
3. Leave the **"To"** field empty OR add your own email address
4. **Recommended:** Put your own email in "To" field to track delivery

## ğŸ†˜ Troubleshooting

### Email looks broken after pasting?

- Make sure you selected ONLY the email body content
- Don't use Ctrl+A - use mouse selection
- Try pasting in a new compose window
- Check if signature was removed before pasting

### Email going to spam?

- Send in smaller batches (5-10)
- Increase wait time between batches (30 min)
- Add a personal line at the top like "Dear [Name],"
- Avoid spam trigger words in subject line

### Images not showing?

- This is normal for first view in some email clients
- Recipients need to click "Display images" once
- All images are properly hosted and will load

### Formatting lost?

- Re-copy from the original template email
- Make sure you're pasting in HTML mode (not plain text)
- Gmail sometimes auto-converts - try again
